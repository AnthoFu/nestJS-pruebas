# Comandos generales

1. Instalar paquetes
```bash
npm install
```
2. Correr el proyecto en modo dev

```bash
npm run start:dev
```

3. Ayuda de comandos de nest
```bash
nest -h
```

4. Generar un nuevo recurso 
```bash
nest g resource {nombre aqui} --no-spect 
```
> (el --no-spect es para evitar que genere archivos de test)
